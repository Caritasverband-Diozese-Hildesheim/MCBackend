Module: modules/app

Module that prepares the application

Tells the Server what middleware we use and we set our routes

Middleware we use:

  • body-parser : reads the body of a request from a browser and parses it into something we need. F. e. json and URL encoded
  • method-override : Lets you use HTTP verbs such as PUT or DELETE in places where the client doesn't support it.
