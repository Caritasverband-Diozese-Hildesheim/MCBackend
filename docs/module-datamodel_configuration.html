Name Type Description
scheme string | "http:// or https://" is it secure HTTP or not.
host string | "IP-Adress or 'localhost'" IP-Address or "localhost" to start the server on
port number | "only positives from 1024 til 65534" Port to start the server on
externalUrl string | "has to be an URL" If the external URL differs the baseUrl, we need to define it. MUST be set with environment variable
baseUrl string | "has to be an URL" Root URL of the app. Automatically generated from scheme, host, port
DMSUrl string | "has to be an URL" URL for DMS we want to work with. MUST be set with environment variable
DMSUserEmail string | "EMail" Login Email for the DMS MUST be set with environment variable
DMSAPIToken string | "min 20 Chars, max 60 Chars" API Token for the DMS. MUST be set with environment variable
OIDCAuthUrl string | "has to be an URL" Root URL of the OID Server. MUST be set with environment variable
OIDCRealm string | "min 4 Chars, max 60 Chars" Server Realm-Name of the OID Server. MUST be set with environment variable
OIDCCientId string | "min 4 Chars, max 60 Chars" Client ID to identify with on the OIDC Server. MUST be set with environment variable
OIDCSecretToken string | "min 20 Chars, max 60 Chars" Secret Token to identify with on the OIDC Server. MUST be set with environment variable
OIDCRedirectUrlCallback string | "has to be an URL" URL to be redirected for the OIDC Callback MUST be set with environment variable
OIDCRedirectUrlLogout string | "has to be an URL" URL to be redirected after logout MUST be set with environment variable
